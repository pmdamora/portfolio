Projects

Some of my recent work and projects can be found in their entirety on my Github page. Additionally, a select few of my projects are detailed below.

   rails-weather : demo - A Ruby on Rails application which uses a public weather api to allows users to track the weather conditions and forecasts in different cities.
   fnv-cracker-app : demo - A Java FNV-1 128 bit brute-force hash cracker web application built with Spring Boot.
   SoundArt - SoundArt is a Flask web application to create waveform images from an uploaded audio file.
   Forest - A Python, Flask, Keras, and OpenCV project that leverages machine learning, feature extraction, and image processing techniques to predict the species of a leaf.
   CourseApp - Submission for the 2014 UVM CS Fair: A Python web application to help students choose courses
   ccby-merger - A simple CLI built with Node.js to make the process of adding copyright information to end of lecture videos and other course content significantly faster.
   tictactoe - I created a version of the classic game of Tic-Tac-Toe in order to improve my knowledge and understanding of PyQt and the Qt framework, as well as simply gain more experience creating non-web UIs.
   ratemyslumlord - Submission for 2nd annual UVM/State Street Codefest: A simple Python website to help people rate their landlords and aggregate renting data.
   TravLog - Travlog is a mobile app developed for the iPhone which monitors the location of participants in a travel survey. The app is designed to gather information regarding the mode, purpose and frequency of intercity travel to inform policy makers and provide detailed information to aid infrastructure planning.
